dotfiles
========

Useage
------
Install dependencies.

    $ sudo aptitude install git vim

Clone the repository.

    $ cd

    $ mkdir Projects && cd Projects

    $ git clone https://github.com/tdkn/dotfiles.git

Make as a symlink.

    $ cd

    $ ln -s ~/Projects/dotfiles ~/.dotfiles

Execute bootstrap.sh to setup.

    $ cd .dotfiles

    $ source bootstrap.sh
